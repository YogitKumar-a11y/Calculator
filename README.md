# 🧮 Modern Web Calculator

A sleek, responsive, and fully functional web-based calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- **Modern UI:** Clean dark mode design.
- **Responsive Layout:** Works perfectly on mobile screens, tablets, and desktops.
- **Backspace Functionality:** Easily correct typos with the delete (⌫) button.
- **Smart Math:** Handles floating-point precision bugs and prevents division-by-zero errors.

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## 💻 How to Run This Project
You don't need to install anything! Just download or clone this repository and open the `index.html` file in any web browser to use the calculator instantly.
